# Funnel Launch Checklist

## Pre-Build

- [ ] Dream Buyer Profile completed (demographics + psychographics)
- [ ] Dream 100 list drafted (where do they hang out?)
- [ ] Lead magnet created (solves ONE specific problem)
- [ ] Value ladder mapped (free → tripwire → core → premium)
- [ ] Guarantee / risk reversal defined
- [ ] Maximum allowable CAC calculated

## Build

- [ ] Opt-in page live (headline, bullets, form, CTA)
- [ ] Thank-you page with tripwire / next step
- [ ] Sales page (if selling directly)
- [ ] Checkout page with upsell
- [ ] Confirmation / thank-you page
- [ ] All pages mobile-responsive (test at 320px)
- [ ] Page load time under 3 seconds
- [ ] No broken links or missing images

## Email

- [ ] Welcome email (immediate, delivers lead magnet)
- [ ] Soap Opera Sequence (5 emails, days 1-5)
- [ ] Seinfeld Sequence planned (ongoing, 2-3/week)
- [ ] Cart abandonment email (if applicable)
- [ ] Godfather Sequence for unconverted leads (days 7-14)
- [ ] Breakup email (day 14)
- [ ] All email links tested and working
- [ ] Unsubscribe link visible in every email
- [ ] SPF/DKIM/DMARC configured for sending domain

## Tracking

- [ ] GA4 installed on all funnel pages
- [ ] Conversion events configured (lead, checkout, purchase)
- [ ] UTM parameters on all traffic sources
- [ ] UTM persistence across funnel (sessionStorage)
- [ ] Revenue tracking connected (Stripe, PayPal, etc.)
- [ ] Funnel metrics dashboard set up
- [ ] Facebook/LinkedIn pixel installed (if using paid social)

## Legal

- [ ] Privacy policy linked on opt-in pages
- [ ] Terms of service linked on checkout pages
- [ ] GDPR consent checkbox (if targeting EU)
- [ ] CAN-SPAM compliant unsubscribe mechanism
- [ ] Refund/guarantee policy clearly stated

## Launch

- [ ] Full funnel walkthrough completed (opt-in → purchase → email)
- [ ] Test payment processed and refunded
- [ ] All emails received and links working
- [ ] Mobile walkthrough completed
- [ ] 404 page configured
- [ ] Initial traffic source activated
- [ ] Day-1 metrics baseline recorded
- [ ] A/B test plan drafted (offer first, then headline)

## Post-Launch (Week 1)

- [ ] Daily metrics review (EPL, conversion rates, CAC)
- [ ] Email deliverability check (inbox vs. spam)
- [ ] Cart abandonment rate reviewed
- [ ] First A/B test launched
- [ ] Customer feedback collected (reply to first 10 buyers)
